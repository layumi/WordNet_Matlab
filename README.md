## WordNet_Matlab
A simple api for matlab to search semantic synonym

1.Download WordNet by nltk (a python package)

`pip install nltk` or `conda install nltk`

`python>> import nltk
python>> nltk.download()`

2.Run the code

`matlab>> synsets('boy');
matlab>> ans = 

    ''male_child.n.01''    ''boy.n.02''    ''son.n.01''    ''boy.n.04''`
